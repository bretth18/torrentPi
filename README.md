# torrentPi
>client for streaming torrents on pi
>>built with nodejs, express, peerflix

##installing
- run ```sudo npm install```
- to start: ``` sudo node server.js```
- web interface available at http://localhost:1337

##requirements
- VLC Player: ```sudo apt-get install vlc```
- nodejs, npm
- a raspberry pi
